5 Html Tags

a:
The tag is used to create hyperlinks to other web pages, files, or locations on the same page, it helps to make coding really easy.
sub
This makes a subscript
h1 to h6:
The headings tag h1 to h6 is used to define headings and subheadings within the document, the headings can range from h1 to any number and it helps html organise your work better.
body:
The body tag is where the main content/information of the html document is placed which is visible to the user.
ul:
This tag creates an unordered list in the html document which is regarded as the bulleted list. It is used together with the (li)tag to create an unordered list
